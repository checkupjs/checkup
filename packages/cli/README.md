# Usage

<!-- usage -->

```sh-session
$ yarn global add @checkup/cli

$ checkup COMMAND
running command...

$ checkup (-v|--version|version)
@checkup/cli/0.0.0 darwin-x64 node-v10.18.0

$ checkup --help [COMMAND]
USAGE
  $ checkup COMMAND
...
```
